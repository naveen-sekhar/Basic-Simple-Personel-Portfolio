Smart Parking System

The Smart Parking System is built using a Raspberry Pi 4 and a camera module to capture car number plates and dynamically allocate parking slots. It includes:

  • Real-time license plate recognition using OpenCV.
  • Entry and exit time logging.
  • Dynamic slot management with a CSV-based database.
  • Trained model deployed on Raspberry Pi.

Technologies Used: Python, OpenCV, Raspberry Pi, CSV, Camera Module
